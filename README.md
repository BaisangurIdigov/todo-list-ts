# TodoList приложение 

Это личный эксперементальный проект сделанный после изучения TypeScrypt

## Функционал

* Добавление дела
* Удаление дела
* Обозначение выполненного дела

## Запуск проекта

Для запуска проекта вам необходимо набрать команду в терминале:

```sh 
npm i
```

После набрать команду:

```
npm start
```
